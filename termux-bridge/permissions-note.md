# Permissions note

After cloning in Termux, make the scripts executable manually if needed:

```bash
chmod +x bridge.py telegram_bot.py start.sh stop.sh test.sh run-telegram.sh direct-am-test.sh termux-boot-start.sh termux-setup-permissions.sh
```

Useful commands:

```bash
./direct-am-test.sh "how much battery left"
./start.sh
./test.sh
./stop.sh
```
